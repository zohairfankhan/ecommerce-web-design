# 🛒 E-Commerce Website (HTML, CSS, JavaScript)

A modern, responsive e-commerce website built using **HTML5, CSS3, and JavaScript**.  
This project was developed as part of a 3-task frontend assignment focusing on UI design, responsiveness, and layout implementation.

**Name:** Zoha Irfan

## 📌 Project Overview

This project includes a complete e-commerce frontend with:

- Responsive top navigation bar
- Search bar with category filter
- Product listing page with grid/list view
- Sidebar filters (price, category, rating, etc.)
- Product cards with image, price, rating
- Pagination system
- Newsletter subscription section
- Fully responsive footer
